## 📋 Contents

지원하는 플랫폼은 아래와 같음

* NAVER
* Google
* KAKAO
* GitHub

OAuth2.0 Backend와 연동되어 기본적인 OAuth2.0 기능을 제공한다.

* 플랫폼 로그인
* 정보 제공 동의 갱신
* 사용자 정보 호출
* 연동 해제

제공하는 기능은 위와 같다.

Frontend의 경우, OAuth2.0 Backend API의 활용 예시를 위해 개발된 것으로, 최대한 간결하고 기본적인 방식으로 구현함.
